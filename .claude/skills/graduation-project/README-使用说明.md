# 毕业设计开题报告工具包

## 📁 文件说明

1. **01-开题报告模板.md**
   - 完整的开题报告结构模板
   - 包含所有必需章节
   - 直接填充内容即可使用

2. **02-写作指导.md**
   - 各部分详细写作要点
   - 常见错误和技巧
   - 示例和注意事项

3. **03-格式检查清单.md**
   - 提交前的完整检查清单
   - 格式规范要求
   - 评分标准参考

## 🚀 使用步骤

1. 打开「01-开题报告模板.md」
2. 参考「02-写作指导.md」填写各部分内容
3. 完成后用「03-格式检查清单.md」逐项检查
4. 导出为 PDF 或 Word 提交

## 📌 注意事项

- 本模板为通用格式，请根据学校具体要求调整
- 建议尽早开始，预留充足修改时间
- 多与导师沟通，及时获取反馈
