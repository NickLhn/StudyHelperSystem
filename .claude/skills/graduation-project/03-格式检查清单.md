# 开题报告格式检查清单

## 📋 提交前检查

### 基本信息
- [ ] 论文题目准确、简洁（一般不超过 25 字）
- [ ] 学生信息完整（姓名、学号、专业、班级）
- [ ] 指导教师信息完整
- [ ] 学院、日期填写正确

### 内容完整性
- [ ] 选题背景与意义
- [ ] 国内外研究现状
- [ ] 研究目标与内容
- [ ] 研究方法与技术路线
- [ ] 创新点
- [ ] 预期成果
- [ ] 研究进度安排
- [ ] 参考文献
- [ ] 指导教师意见（留空）
- [ ] 评审意见（留空）

### 格式规范
- [ ] 字体：中文宋体/黑体，英文 Times New Roman
- [ ] 字号：正文小四，标题适当加大
- [ ] 行距：1.5 倍行距或固定值 20 磅
- [ ] 页边距：上下 2.54cm，左右 3.17cm（Word 默认）
- [ ] 页码：从正文开始编号
- [ ] 图表：有编号和标题，引用正确

### 文字质量
- [ ] 无错别字
- [ ] 无语病
- [ ] 语句通顺
- [ ] 专业术语使用准确
- [ ] 标点符号正确

### 参考文献
- [ ] 数量不少于 15 篇
- [ ] 外文文献不少于 5 篇
- [ ] 近 5 年文献占 50% 以上
- [ ] 格式符合 GB/T 7714 标准
- [ ] 正文引用与参考文献列表对应

### 图表质量
- [ ] 图表清晰
- [ ] 编号连续
- [ ] 标题准确
- [ ] 正文中有引用说明

---

## 🔍 自查问题

1. 我的研究问题是否明确？
2. 研究内容是否具体可行？
3. 研究方法是否适合解决研究问题？
4. 创新点是否有充分依据？
5. 进度安排是否合理？
6. 是否阅读了足够的文献？
7. 是否有明显的逻辑漏洞？

---

## 📊 评分标准参考

| 项目 | 权重 | 评分要点 |
|------|------|----------|
| 选题质量 | 15% | 意义、value、难度 |
| 文献综述 | 20% | 全面性、准确性、时效性 |
| 研究内容 | 25% | 目标明确、内容具体、重点突出 |
| 研究方法 | 20% | 科学性、可行性、创新性 |
| 进度安排 | 10% | 合理、可行 |
| 格式规范 | 10% | 符合学校要求 |
